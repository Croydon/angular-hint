Exclusive Hint Example
This is a test div that the controller will attempt to remove. Such manipulation of the DOM will cause ng-hint-dom to log a message if it is loaded correctly by the ng-hint-exclude tag. Only ng-hint-directives is listed in the ng-hint-exclude. Therefore, all other modules should be loaded.
